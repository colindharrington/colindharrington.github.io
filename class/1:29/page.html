Paletton

http://paletton.com/#uid=1000u0kllllaFw0g0qFqFg0w0aF

  • This is a website designed to help with color pairing
  • The website has a color wheel and a display section where the colors are shown. The display section gives a few variations on the selected color tone.

I think this website has a very intuitive navigation. The websites main use is right on the front page, and you don't have to scroll or click through to get to the utility.

Met

https://www.metmuseum.org/art/collection/search#!?searchField=All&showOnly=openAccess&sortBy=relevance&offset=0&pageSize=0

  • This website hosts a large image collection of the various artworks owned of housed by the met.
  • The interface makes sense, and does a good job of emphasizing the images and their ability to be arranged and examined.

This website has very appropriate navigation. The image resource is a catalogue, and most of the features on the website are there to display information or highlight images.

A third website

https://nobodyhere.com/nobodyhere/justme/

  • This website is an art project.
  • On the "Why?" section of the page, the author explains "I like to express myself using animation, text and programming. The site has been on-line since 1998 and grew into hundreds of pages."

The navigation for this page is very abstract. Given the contents of the website, this navigation is very effective because it draws you in and keeps you curiously exploring.
